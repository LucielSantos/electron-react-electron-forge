# Electron + React.js template

Este template foi criado a partir do template de Typescript + Webpack do electron forge.

---

Este template inclui:

- Electron
- Electron forge
- React.js
- Typescript
- Webpack
- ESLint
- Styled-components
- React router dom
- Configuração entre renderer e main
- Electron forge
- Configuração de banco de dados com:
  - Sqlite3
  - Typeorm
